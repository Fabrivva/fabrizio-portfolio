My Portfolio

How I built this website from scratch using Claude Code

Step 01

The Challenge

Transitioning into Product Management without a formal PM title requires demonstrating product instincts through tangible artifacts. A traditional CV cannot convey systems thinking, user empathy, or prioritisation skills Instead of using a template, I started from scratch, writing basic line of HTML and CSS with the help of Claude Code.

I sketched the structure around three pillars: who I am, what I've built, and how to reach me. Clean, minimal, and fast, no frameworks, no bloat.

Step 02

Strategy and PRD

Define how am I going to develop the website. Using Claude I have asked to define a PRD based on my challenge.

The workflow: describe the intent, review the output, refine. No copy-pasting from Stack Overflow, no boilerplate plugins. Just a clear product vision translated into working code, one component at a time.

Step 03

Choose the correct AI tools

I’ve decided to move forward with Claude and Claude Code. The reason is simple: although I already have coding skills, using Claude Code within VS Code saves me a huge amount of time and helps improve my code cleanliness.

Step 04

Deploy and Launch Product

Using Claude Code I created a prompt to connect the deploy the website with Github and published it.
